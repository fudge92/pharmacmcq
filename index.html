Topic file format

One JSON file per topic, then list all topic files inside topic-files.json.
{
  "title": "Topic 1",
  "questions": [
    {
      "id": "t1-q1",
      "question": "Which of these are prime numbers?",
      "options": ["2", "4", "5", "9"],
      "correctAnswers": [0, 2],
      "explanation": "2 and 5 are prime."
    }
  ]
}
{
  "files": ["topic-math.json", "topic-science.json"]
